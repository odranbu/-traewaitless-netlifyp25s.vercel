# WAITLESS - Queue Management System

A professional queue management system website showcasing 16+ business solutions.

## Features

- **Hero Section** - Prominent hero image with call-to-action for 5-day free trial
- **Statistics Dashboard** - Key metrics showing business value
- **16+ Business Solutions** - Comprehensive coverage of various industries:
  - Healthcare (Pharmacy, Hospital, Veterinary, Dental)
  - Personal Care (Barber, Beauty Salon)
  - Hospitality (Restaurant, Hotel, Coffee Shop)
  - Government Services
  - Financial Services (Bank)
  - Automotive (Gas Station/Auto Service)
  - Retail
  - Fitness (Gym)
  - Professional Services (Law Office)
  - Education & Career Services
  - Entertainment (Theater/Movies)

## Deployment

This is a static website optimized for Vercel deployment. All assets are in the root directory for simple deployment.

### Structure
```
/
├── index.html           # Main homepage
├── vercel.json         # Vercel configuration
├── favicon.png         # Site favicon
├── favicon.svg         # SVG favicon
├── hero.png           # Hero section image
└── [business images]  # All business type images
```

## Technologies

- Pure HTML5/CSS3
- Responsive design
- No build process required
- Optimized for performance
